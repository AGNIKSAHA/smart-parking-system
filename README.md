# Smart Parking Management System

Production-oriented full-stack system for malls, corporate offices, and residential societies.

## Stack
- Backend: Node.js, Express, TypeScript, MongoDB, Socket.IO, Stripe, JWT cookies.
- Frontend: React, TypeScript, TanStack Query, React Router v7, React Hook Form, Tailwind.

## Setup
1. Copy `backend/.env.example` to `backend/.env` and fill secrets.
2. Install dependencies from project root:
   - `npm install`
3. Run both apps:
   - `npm run dev`

## API Base
- Backend: `http://localhost:5000/api/v1`
- Frontend: `http://localhost:5173`
